# Velvet Cafe Website

## Overview

Velvet Cafe is a simple front-end website that simulates an online cafe platform where users can log in, browse the menu, and manage their cart. The project focuses on basic web development using HTML, CSS, and JavaScript.

## Features

* Login page for users
* Option to continue as a guest
* Navigation bar with Home, Menu, and Cart pages
* Hero section with promotional text
* External Google Fonts integration
* Separate styling using CSS
* JavaScript functionality for login and navigation

## Project Structure

```
Velvet-Cafe/
│
├── index.html      # Main page with login and home sections
├── menu.html       # Menu page displaying cafe items
├── cart.html       # Shopping cart page
├── style.css       # Website styling
├── script.js       # JavaScript for login and navigation
└── README.md       # Project documentation
```

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts (Poppins)

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in a web browser.
4. Use the login form or continue as a guest to enter the site.
